# BUGFIX - 긴급 수정 필요 항목

## 발견된 버그 목록

### BUG #1: 미리보기에 빈 항목 2가 항상 표시됨
- **현상**: 값 입력 안 해도 항목 2가 미리보기에 표시
- **원인**: SimpleQuote에서 빈 항목도 필터링 없이 렌더링
- **해결**: description이 비어있는 항목은 렌더링에서 제외

### BUG #2: 비율 가산(percent) 로직 미구현
- **현상**: percent 타입 선택해도 실제 계산 안 됨
- **원인**: calculateItemTotal()에서 percent 타입 처리 없음
- **해결**: 제작비(페이지비용) 기준으로 % 계산 로직 추가

### BUG #3: 서버/도메인 확정 시 항목에 미표시
- **현상**: 서버/도메인 "확정" 선택해도 견적 항목에 추가 안 됨
- **원인**: generateEstimateQuoteItems()에서 서버/도메인 항목 생성 로직 없음
- **해결**: 서버/도메인 확정 시 해당 비용을 항목으로 자동 생성

### BUG #4: 비고란 작성 시 레이아웃 깨짐
- **현상**: 비고 내용이 길어지면 추가옵션 안내/푸터가 밀려남
- **원인**: 비고란이 flex-grow 영역에서 높이 제한 없음
- **해결**: 비고란 최대 높이 제한 + overflow 처리, 또는 레이아웃 재구성

### BUG #5: 비고란 레이아웃 변경 요청
- **요청**: 비고란을 항목란처럼 한 줄 가로로 표시
- **해결**: 비고 영역을 테이블 항목과 동일한 그리드 레이아웃으로 변경

---

### BUG #6: 간단 견적서 하단 푸터 제거
- **요청**: 간단 견적서 하단 푸터 불필요
- **해결**: 푸터 영역 완전 제거
