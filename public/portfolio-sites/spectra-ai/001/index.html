FAQ

자주 묻는 질문

모든 데이터는 AES-256 암호화로 저장되며, 전송 시 TLS 1.3을 사용합니다. SOC 2 Type II 인증을 취득했으며, 매년 독립 보안 감사를 진행합니다. 고객 데이터는 한국 리전(AWS ap-northeast-2)에 저장됩니다.

기본 설정은 30분 이내에 완료됩니다. 전담 온보딩 매니저가 데이터 소스 연동부터 첫 대시보드 구성까지 1:1로 안내합니다. 대부분의 팀이 3일 안에 본격적인 사용을 시작합니다.

developers.spectra-ai.co.kr에서 전체 API 문서를 확인하실 수 있습니다. RESTful API, GraphQL 엔드포인트, Webhook 설정 가이드, SDK(Python/JS/Go) 문서가 포함되어 있으며, Postman 컬렉션도 제공합니다.

신용카드 없이 14일간 Growth 플랜의 모든 기능을 무료로 사용하실 수 있습니다. 체험 기간 중에도 전담 온보딩 지원이 제공되며, 체험 종료 후 자동 결제는 발생하지 않습니다.
