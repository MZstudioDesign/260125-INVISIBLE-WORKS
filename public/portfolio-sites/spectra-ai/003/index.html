도입 전 궁금한 점을 확인하세요.

스펙트라 AI는 SOC 2 Type II 인증을 획득했으며, 모든 데이터는 AES-256 암호화로 전송 및 저장됩니다. GDPR 및 국내 개인정보보호법을 완벽히 준수하며, 정기적인 모의 해킹 테스트를 실시합니다.
전담 온보딩 매니저가 배정되며, 평균 2주 이내에 핵심 대시보드 구축이 완료됩니다. 기존 데이터 소스 연결부터 팀 교육까지 전 과정을 지원합니다.
RESTful API와 GraphQL 엔드포인트 모두 지원하며, 상세한 API 레퍼런스 문서와 SDK(Python, Node.js, Java)를 제공합니다. Swagger UI 기반 인터랙티브 문서도 포함됩니다.
모든 요금제에 14일 무료 체험이 포함되어 있으며, 신용카드 없이 시작할 수 있습니다. Growth 플랜은 신청 시 30일까지 연장 가능합니다.
Slack, Jira, Google Sheets, Salesforce 등 주요 비즈니스 도구와 네이티브 연동을 지원합니다. 커스텀 데이터 소스는 REST API 또는 Webhook으로 연결할 수 있습니다.
